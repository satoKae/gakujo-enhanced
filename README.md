## 機能

- 学情全体のカラーテーマを変更します
- サイドバーを追加します
- ホーム画面の成績を自動で隠します
- スケジュールページで月間表示のときカレンダーのダウンロードボタンを追加します Googleカレンダーなどに時間割をインポートできます
- 授業参考情報一覧の教科一覧を並び替えます
- 連絡一覧の表示件数を50件に設定します
- 学情のログイン画面で自動でログインを試行します

## インストール

ビルドには[Bun](https://bun.com/)が必要です。

1. リポジトリをクローン

    ```bash
    git clone https://github.com/satokae/gakujo-enhanced.git
    cd gakujo-enhanced
    ```

2. 依存のインストール

    ```bash
    bun install
    ```

3. ビルド

    ```bash
    bun run build
    ```

`dist/index.user.js` にファイルが生成されます。

## 開発

`bun run dev` で開発サーバーが起動します。
