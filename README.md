## 機能

- 学情全体のカラーテーマを変更します
- サイドバーを追加します
- ホーム画面の成績を自動で隠します
- スケジュールページで月間表示のときカレンダーのダウンロードボタンを追加します Googleカレンダーなどに時間割をインポートできます
- 授業参考情報一覧の教科一覧を並び替えます
- 連絡一覧の表示件数を50件に設定します
- 履修科目選択の表示件数を50件に設定します
- 学情のログイン画面で自動でログインを試行します
- シラバス・連絡詳細ページに貼られているリンクをクリックできるようにします
- 連絡一覧ページにおいて、既読としたレポート登録通知を隠します
- キーボードショートカットを追加します
  - `Ctrl` + `I`: お知らせ一覧のポップアップ (右上の i マーク) を開閉します
  - `[`: サイドバーにフォーカスを移動します (サイドバー機能が有効な場合)
  - `]`: main 要素の中にフォーカス可能な部分がある場合はそこへフォーカスを移動します
- tab でフォーカスを移動しているとき、隠れているポップアップがフォーカスされないようにします

## インストール

ビルドには[Bun](https://bun.com/)が必要です。

1. リポジトリをクローン

   ```bash
   git clone https://github.com/satokae/gakujo-enhanced.git
   cd gakujo-enhanced
   ```

2. 依存のインストール

   ```bash
   bun install
   ```

3. ビルド

   ```bash
   bun run build
   ```

`dist/index.user.js` にファイルが生成されます。

## 開発

`bun run dev` で開発サーバーが起動します。
